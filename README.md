# react-native-barcode-view

React Native barcode rendering view

## Installation

```sh
npm install react-native-barcode-view
```

## Usage

```js
import { BarcodeViewView } from "react-native-barcode-view";

// ...

<BarcodeViewView color="tomato" />
```
